# MADDER(Akane)

A Video Editor for the Next Generation

## Requirements

- Rust
- Gstreamer >= 1.10

```sh
$ sudo apt install -y libglib2.0-dev libgstreamer1.0-dev libpango1.0-dev libgdk-pixbuf2.0-dev libgstreamer-plugins-bad1.0-dev libgtk-3-dev
```
